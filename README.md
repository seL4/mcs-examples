# MCS-scheduling examples

To run the native example:
`../init-build.sh -DPLATFORM=pc99`

To run the CAmkES example:
`../init-build.sh -DPLATFORM=pc99 -DCAMKES_APP=1`

Currently x86 and arm supported.
